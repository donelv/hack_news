# Интерфейс для сайта Hacker News, состоящий из двух страниц, с использованием React & Redux

## Запуск приложения
1. Скачайте файлы из ветки main
2. Установите необходимые зависимости командой:
#### `yarn install`
3. Запустите приложение командой: 
#### `yarn start`
